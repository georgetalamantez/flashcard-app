# Flashcard App - Local AI Study Tool

This application is designed to help users study complex technical and medical topics using AI-generated flashards.

## Features
- Local execution for privacy.
- Support for multiple flashcard sets (JSON format).
- Simple web Interface for study sessions.

## Project Structure
- `json/`: Contains flashcard data sets.
- `convert_flashcards.py`: Utility script to convert Markdown flashcards to JSON.
- `backend/`: Python backend logic.
- `frontend/`: Web frontend components.
